# 4Help

<h2>Setup</h2>

<ul>

    Node >= 10.x

    yarn >= 1.10.x 

    npm >= 6.x 

</ul>

<h2>Run Project</h2>

Enter the project server folder:  ```yarn``` or ```npm install``` and after ```yarn dev``` or ```npm run dev```

<h2>Routes</h2>

Route => ```http://localhost:3333``` this return <b>Welcome to 4Help</b>

You can import file of <b>Insomnia</b> => 'insomnia.json'.
